# Caesar cipher CLI

### Arguments
* -s, --shift: a shift
* -i, --input: an input file
* -o, --output: an output file
* -a, --cli-action: an action encode/decode
  

### Usage example:

```
$ node cli -s 7 -i "./input.txt" -o "./output.txt" -a encode
$ node cli --shift 7 --input input.txt --output output.txt --action decode
```
